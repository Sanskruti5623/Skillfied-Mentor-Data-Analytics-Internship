## 🏦 Task 1: Bank Marketing Data Analysis
📌 Project Overview
This project analyzes the bankmarketing.csv dataset to understand customer behavior and term deposit subscription patterns.
The main goal was to inspect the dataset and explore factors affecting whether a customer subscribes to a term deposit.
🔍 Work Performed
1️⃣ Data Loading & Inspection
Loaded dataset using Pandas
Checked data types
Verified missing values
Used .info() and .describe() for summary statistics
2️⃣ Data Cleaning
Checked for null values
Verified overall dataset consistency
3️⃣ Exploratory Data Analysis (EDA)
Distribution of target variable (y)
Count plots for term deposit subscription
Percentage analysis of subscription vs non-subscription
Visualizations using Seaborn & Matplotlib

🛠 Tools Used
Python
Pandas
Matplotlib
Seaborn

🎯 Outcome
Understood customer subscription distribution
Identified dataset structure and feature types
Prepared foundation for predictive modeling
